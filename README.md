# parser
a JavaScript module for parsing [jsig](https://github.com/jsigbiz/spec)

## install
```console
$ npm install jsig
```

## example

``` js
var parse = require('jsig')
var ast = parse('reduce: (array: Array, (accumulated: Any, elem: Any) => Any, seed: Any) => Any')
console.log(ast, null, 2)
```
